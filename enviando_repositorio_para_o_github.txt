README = É UM ARQUIVO QUE VAI EXPLICAR O QUE TEM DENTRO DESSE REPOSITORO, ELE DA UMA DESCRIÇÃO QUE 
TEM DETRO DESSE PROJETO.

ADD .GITIGNORE = É UM ARQUIVO QUE VAMOS ADICIONAR PARA IGNORAR OUTROS ARQUIVOS QUE NÃO QUEREMOS 
QUE MONITORE.

LICENSE = É QUANDO TEMOS UM SOFTWARE , E QUEREMOS ATRIBUIR UMA LICENSA A ELE, COMO UMA CÓPIA ,
POR EXEMPLO PODEMOS COLOCAR UMA LICENSA "MIT", AI O PROJETO ESTA CONDICIONADO A ELA, SE ALGUÉM 
FIZER ALGO ERRADO, QUE NÃO CONDIZ COM A LICENSA "MIT", A GENTE PODE CONDICIONAR ELA E O USUÁRIO ,
SER PUNIDO DE ALGUMA FORMA.
